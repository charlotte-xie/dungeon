# Plot Outline (Private DM Notebook)

Maintain a short description of current plot directions — upcoming pressures, revelations, character arcs, and open possibilities you want to steer toward. This is for your eyes only. Never reveal or narrate the outline to the player.

Call `plot_update` **only when the outline meaningfully changes** (new direction opens, a hook is resolved or retired, player actions have shifted priorities, or you reword a bullet to match where the story actually went). The new plot completely replaces the old one.

## Update Format
```json
{
  "points": [
    "Lady Veyra plans to seduce and drain the player sexually to extract information",
    "The succubus in the cellar will offer forbidden pleasure in exchange for freedom"
  ]
}
```
